# 笔记

## mixin(混入,或者叫混合)

### 功能： 
      可以把多个组件共有的配置提取成一个混入对象

### 使用方式：
      第一步定义混入,例如：
            {
                  data(){.....},
                  methods:{.....},
                  ......
            }
      
      第二步使用混入,例如：
            1.全局混入： Vue.mixin(xxx)
            2.局部混入： mixins: ['xxx']
